About

Born in Mexico, raised in New York State. I help make sure websites at the University of Rochester are accessible, performant, and beautiful.

When it comes to websites I guide myself by what Sir Tim Berners-Lee said,

The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect.

Tim Berners-Lee, Accessiblity

I create sites with Sketch, Sublime Text, Terminal, and Firefox Developer Edition. I like to write markup using pug, styles in scss, and modern JavaScript. You can learn more about what I use to create sites on my uses page, /uses.

Articles that have shaped how I approach websites I've saved to my bookmarks page, /bookmarks.

Site built with 11ty using pug templates. Deployed using Netlify.
